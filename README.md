# amigo_secreto
Sistema para realizar sorteio de amigo secreto! o/

# Como instalar?

Crie um virtualenv:

```
python3.6 -m venv .venv
```

Ative o virtualenv:

```
source .venv/bin/activate
```

Instale os requisitos:

```
pip install -r requirements.txt
```

Inicie o projeto:

```
python manage.py runserver
```
